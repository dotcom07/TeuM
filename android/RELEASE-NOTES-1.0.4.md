# 틈새움 1.0.4

## 한국어 출시 노트

하단 시스템 버튼과 겹치지 않도록 화면 배치를 개선했어요.

- 기기별 하단 안전 영역을 반영해 시간 선택과 알람 멈춤 화면의 버튼이 가려지지 않아요.
- 3버튼 내비게이션을 쓰는 기기에서 하단 바가 앱의 다크 네이비 톤과 자연스럽게 이어져요.

## English release notes

Improved layouts around the Android system navigation area.

- Time selection and pause controls now stay above device navigation buttons.
- The bottom system area better matches TeuM’s dark navy interface on three-button navigation devices.
