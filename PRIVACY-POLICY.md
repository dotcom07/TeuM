# 틈새움 TeuM 개인정보처리방침

시행일: 2026-07-21

공개본: [Notion 개인정보처리방침](https://aluminum-language-c6a.notion.site/TeuM-3a447d4c6bf180f7a6f6e221089a5012?pvs=73)

틈새움 TeuM(이하 “앱”)은 직장인이 업무 중 짧은 건강 습관을 실천하도록 돕는 온디바이스 리마인더입니다.

## 1. 수집·전송하는 정보

앱은 개인정보, 건강 데이터, 위치 정보, 연락처, 기기 식별자, 사용 기록을 서버로 수집하거나 제3자에게 전송하지 않습니다.

앱이 동작을 위해 기기에만 저장하는 정보는 다음과 같습니다.

- 업무 시작·종료 시간, 반복 요일, 알람 간격
- 알람 켜짐 여부와 무음/진동 방식
- 다음 알람 시각과 일시정지 상태
- 사용자가 `기록하며 사용`을 선택한 경우: 각 회차의 `챙겼어요`, `이번 틈은 넘겼어요`, 선택 없이 닫은 상태와 해당 시각

이 정보는 앱 기능 제공을 위해 사용자의 기기에만 저장되며, 개발자나 제3자가 접근할 수 없습니다. `기록하며 사용`은 기본값이며, `가볍게 사용`에서는 실천 내역을 저장하지 않습니다.

사용자가 설정의 `백업 및 기기 간 이동`에서 백업 파일 만들기를 직접 선택한 경우에만, 설정·기록·책상 배치를 하나의 파일로 만들어 사용자가 고른 파일 위치 또는 공유 앱에 전달할 수 있습니다. 이 과정에서 앱은 개발자 서버로 정보를 전송하지 않으며, 파일을 어디에 보관·공유할지는 사용자가 선택합니다.

## 2. 권한 사용

앱은 아래 권한 또는 시스템 접근을 사용할 수 있습니다.

- 알림: 업무 시간 중 리마인더를 표시합니다.
- 정확한 알람 및 알림·리마인더 특별 접근: 사용자가 정한 시각에 리마인더를 예약합니다.
- 진동: 사용자가 진동 알림 방식을 선택한 경우에만 짧은 진동을 제공합니다.
- 전체 화면 알림: 지원 기기에서 사용자가 허용한 경우, 잠금 화면에서도 1분 리마인더를 표시할 수 있습니다. 허용하지 않아도 일반 알림으로 기능을 이용할 수 있습니다.

## 3. 보관과 삭제

설정은 사용자가 앱 데이터를 삭제하거나 앱을 삭제할 때 함께 삭제됩니다. `기록하며 사용`으로 저장한 틈 기록은 앱 설정의 `기록 모두 삭제`로 언제든 영구 삭제할 수 있으며, 앱 데이터 삭제·앱 삭제 시에도 함께 삭제됩니다. 사용자가 만들어 외부에 보관한 백업 파일은 해당 파일 위치에서 직접 삭제해야 합니다. 앱은 별도 계정을 제공하지 않으므로 계정 삭제 절차는 필요하지 않습니다.

## 4. 제3자 제공

앱은 사용자 정보를 제3자에게 판매, 공유, 제공하거나 광고·분석 목적으로 사용하지 않습니다.

## 5. 의료 관련 고지

앱은 의료 서비스, 진단, 치료 또는 건강 상태의 측정 도구가 아닙니다. 사용자는 자신의 상태와 필요한 의료적 판단에 따라 앱을 이용해야 합니다.

## 6. 문의와 변경

개인정보 관련 문의는 Play 스토어에 기재된 개발자 문의 이메일로 보낼 수 있습니다.

정책이 변경되면 이 문서의 시행일을 갱신하고 앱 또는 스토어 공지를 통해 알립니다.
